# Paper Contracts

Contracts are non-negotiable constraints that govern writing this paper. They cannot be overridden by convenience, deadline pressure, or reviewer requests that conflict with them.

## Contract: Literature Review

**Location:** `literature-review/`

One file per reference in `refs.bib`. Each file contains:

- Full bibliographic details and Zotero key
- Source verification status (full text retrieved, abstract only, or metadata only)
- What the source actually establishes (from the text, not from memory)
- Relevance to iterflow — which specific claims the reference supports
- Limitations and caveats — what the reference does NOT establish or where its scope ends
- Explicit list of claims in the paper it supports

**Rule:** No claim in the paper may cite a reference unless that reference's contract file confirms the claim is supported. If the claim is not listed in "Key claims supported", it cannot be cited against that reference without updating the contract file first (which requires re-verifying the source).
